How to run the program:
1. Put the file in a folder in your computer.
2. Run the Command Prompt as an Administrator.
3. In the Command Prompt, change the directory to the folder where Exer1b.java was saved.
4. Type "javac Exer1b.java" to compile the java file.
5. Type "java Exer1b" to run the program.
6. You may input the number of rows and columns for the table to be generated then select 
from the options in the menu provided to manipulate the contents of the table.
You may also type 'x' to exit from the program.
